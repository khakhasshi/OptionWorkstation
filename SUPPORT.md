# Support

Option Workstation is community-supported research software.

Use GitHub issues for reproducible software bugs, documentation gaps,
data-contract questions, and scoped feature proposals. Include the commit,
operating system, browser, Rust and Node versions, mode (replay or live), and
redacted logs.

The project does not provide:

- personalized investment or options advice;
- a signal subscription or performance guarantee;
- support for real-money order submission;
- provider credentials, market-data entitlements, or licensed datasets;
- reimbursement for trading losses, data costs, or infrastructure costs.

Security vulnerabilities must follow `SECURITY.md`. Do not post credentials,
account details, licensed market data, or unredacted screenshots in public.
